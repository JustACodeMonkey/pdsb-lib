AppService

The AppService is configured with the required configuration settings from the PdsbLibModule. These settings are used to get the initial application version and api root.

Properties

apiRoot Returns the API root for the application (typically Angular app name with 'api' appended to the end)
version Returns the application version number
isProd Returns true when running on a production server (gweb11 or ssp-)
isNotProd Returns true when isProd is false
isDev Returns true when running on a dev server (devweb2 or dev-)
isLocalHost Returns true when running on localhost or an IP address with port 4200

Methods

forceRefresh Method to force reload the page
