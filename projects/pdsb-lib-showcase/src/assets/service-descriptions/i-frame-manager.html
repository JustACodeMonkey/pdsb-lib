IFrameManagerService

Use the IFrameManagerService to pass messages between the base application and iFrames.

  • When iFrames are opened this way, logging out of the application will automatically close those children

Properties

parentWindow Returns a reference to the parent window || null
thisFrame Returns a reference to the window of the iFrame || null

Methods

open (url: string, id: string, styleOptions?: {}): void Opens a iFrame
close (id: string, tellParent: boolean = false): void Closes an iFrame
closeAll Closes all open iFrames
postToChild (id: string, message: MessaegData, targetOrigin: string = '*'): void Posts a message to a child iFrame
postToParent (message: MessaegData, targetOrigin: string = '*'): void Posts a message to the Window object of the parent
