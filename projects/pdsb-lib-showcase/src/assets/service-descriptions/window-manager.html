WindowManagerService

Use the WindowManagerSerivce to pass messages between windows and to open child windows.

  • When child windows are opened this way, logging out of the application will automatically close those children

Properties

parentWindow Returns a reference to the parent window || null
thisWindow Returns a reference to this window || null

Methods

open (url: string, name: string): void Opens a window
close (name: string): void Closes a window
closeAll Closes all open windows
postToChild (name: string, message: any, targetOrigin: string = '*'): void Posts a message to a child window
