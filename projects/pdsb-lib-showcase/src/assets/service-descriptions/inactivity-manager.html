InactivityManagerService

The InactivityManagerService is used to log a user out of the application after a set period of time.

Properties

inactiveTimeExceeded Returns the inactiveTimeExceeded Subject. Subscribe to be notified when the inactive time is breached.
secondsUntilWarning (s: number) Sets the number of seconds until the inactive warning is shown. (SET ONLY)
secondsToWarn (s: number) Sets the number of seconds to display the warning. (SET ONLY)

Methods

start Starts the inactivity monitor
stop Stops the inactivity monitor
