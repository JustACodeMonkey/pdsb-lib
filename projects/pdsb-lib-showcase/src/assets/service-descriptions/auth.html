AuthService

The AuthService is used to log in, store and fetch the token, and set HTTP headers.

Properties

displayName Returns the User's displayName
hasToken Returns true when the user has a valid token
headers Returns the HTTP headers with authorization token + any additional required application headers
isLoggedIn Returns true when the user is logged in
isLoggedInAndValidTime Returns true token time is valid AND the user is logged in
loggedOut Returns the loggedOut Subject - Subscribe to be notified when a user logs out
status Returns the User's status
tokenTimeValid Returns true when token's last updated time is valid
userSelected Returns the userSelected Subject - Subscribe to be notified when a user logs in and selects a user account

Methods

addAppHeader (name: string, value: string) Adds a required application header to be added to HTTP requests
getUserProp (prop: string): any Returns the specified User property (or NULL if not found)
login (username: string, password: string, urlPart: string = ''): Observable<IBasicUserInfo[]> Attempts to login and returns a list of IBasicUserInfo
logout (loginId: number, urlPart: string = ''): Observable<IBasicUserInfo[]> Logs out and returns a new IBasicUserInfo
removeAppHeader (name: string) Removes an app header
selectUser (user: IBasicUserInfo) Stores the selected user information
Outputs the .next() event on the userSelected Subject
setUserProp (prop: string, val: any) Sets a User property and updates the stored user variable
updateUserStatus (u?: IBasicUserInfo) Updates the stored user information
