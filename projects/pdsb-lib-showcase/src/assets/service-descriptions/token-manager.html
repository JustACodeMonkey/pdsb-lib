TokenManagerService

Use the TokenManagerService to maintain a token.

  • Tokens expire every 60 minutes on the Java server
  • This service refreshes tokens at 20 minute intervals to ensure it never expires (The 20 minute interval matches the inactivity timeout interval)
  • Tokens are only maintained (updated) when the app is running in standalone mode (Child windows always use the token from the parent window)

Properties

tokenExpired Returns the tokenExpired Subject. Subscribe to be notified when the token has expired.
appVersionChanged Returns the appVersionChanged Subject. Subscribe to be notified when an app version change is detected when refreshing the token.
appHasGoneDown Returns the appHasGoneDown Subject. Subscribe to be notified when an app down change is detected when refreshing the token.
