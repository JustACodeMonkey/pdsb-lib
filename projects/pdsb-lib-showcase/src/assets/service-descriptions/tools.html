ToolsService

Use the ToolsService to display generic alerts, application error alerts, and subscriber errors. Subscriber errors can return a default value.

Properties

appErrorTitle Get/Set the generic app error pop-up title
appErrorMsg Get/Set the generic app error pop-up message
appDownMsg Get the app down message (GET ONLY)
appVersionMsg Get the app version error message (GET ONLY)

Methods

genericError (title: string, msg: string, error?: HttpErrorResponse): MatDialogRef<AlertComponent> Generic error message (single button message)
genericConfirm (title: string, msg: string, btn1Text: string = 'OK', btn2Text: string = 'Cancel'): MatDialogRef<AlertComponent> Displays a generic two-button confirm pop-up
genericChoice (title: string, msg: string, btn1Text: string = 'OK', btn2Text: string = 'Cancel', btn3Text: string = 'Maybe'): MatDialogRef<AlertComponent> Displays a three-button choice confirm box
onAppError (error: HttpErrorResponse): MatDialogRef<AlertComponent> Displays an error using a MatDialog
onSubscriberError (error: HttpErrorResponse, subscriber?: Subscriber<any>, val?: any): MatDialogRef<AlertComponent> Displays an error and completes a subscriber
